## BYR Docs

BYR Docs 是北京邮电大学信息资料分享平台，旨在使校内学生更方便地获取与北邮有关的课程资源与学习生活指南。本站正在运营以下项目：
- [BYR Docs 主站](https://byrdocs.org)，提供课程相关的电子书籍、考试题目和复习资料等；
- [BYR Docs 维基真题](https://wiki.byrdocs.org)，专注于各门课程的近年试题，以可交互、可编辑的呈现形式见长；
- [BUPT 生存指南](https://guide.byrdocs.org)，为学生们提供关于北邮学习、生活方方面面的资料和指引。

若有意参与贡献，请参阅：

- [维基真题编辑者指南](https://wiki.byrdocs.org/guide)；
- [BUPT 生存指南-贡献指南](https://guide.byrdocs.org/contributing/)。
